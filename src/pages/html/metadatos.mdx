## Cabecera 
- Elemento
```html
<tag>contenido </tag>
```
 - Anidacion en orden
```html
<a><b><c>contenido</c></b></a>
```
- Cierre de etriquetas: `<li>` opcional pero recomendable
- Elementos nulos: sin etiqueta de cierre. No anidan contenido (ni texto,  ni tags). Representan imagenes `<img src=''>`, entradas de texto o obotones `<input type=''>`
 y otros elementos  `<col>`, o añaden inforamcion relevante para la pagina `<meta>`,..
- Atributos: se ñaden a la etqueta de apertura. Van separados por espacios
    ```html
    <tag atr1='' atr2=''> contenido </tag>
    ```
