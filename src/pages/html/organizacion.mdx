## Listas 
### Desordenadas
- `<ul></ul>` : contiene una lista desrodenada . Los elementos se muestran con una marca.
- `<li></li>`: cada elemento de la lista. El cierre`</li>`puede omitirse. 

### Ordenadas
- `<ol></ol>`:lista ordenada. Pone delante de cada elemento desde el 1 en adelante por defecto. 
    - **Atributos**:
    - `reversed`: numeración en orden desdcendente (4,3,2,1). `start ="8"`empieza a contar desde el 8.
    - `start`: número asignado al primer elemento a partir del que se cuenta. Siempre numero aunque se muestren numeros romanos. 
    - `type`: **a** letras minusculas | **A** mayusculas | **i** numeros romanos en minusculas | **I** romanos en mayusculas | **1** numeros (por defecto)
- Tambien contiene elementos `<li>`. *Atributos*: `value="34"`  asigna un valor al elemento. Si   los siguientes no llevan value, estos contaran a partir del anterior. Independientemente del `type` llevara un numero siempre. 
### De descripciones 
- `<dl></dl>`: encierran unaa lista de definiciones.
- `<dt></dt>`: termino a describir. Puede estar seguido por ninguna, una o varias descripciones. (Ejemplo : varios sinonimos de un mismo termino)
- `<dd></dd>`: descripcion desarrollada sobre el termino `<dt>`.  
### `<menu>` 
Tiene el mismo comportamiento que una lista desordenada pero se usa semanticamente para denominar un menu.
